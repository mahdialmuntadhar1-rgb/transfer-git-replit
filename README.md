# Iraq Compass 🧭

The first comprehensive Iraqi business directory — 6,955+ real businesses across all 20 governorates.

## Features
- 🌍 3 languages: English, Arabic (RTL), Kurdish Sorani (RTL)
- 🏙️ Filter by all 20 Iraqi governorates
- 📂 9 business categories with live counts
- 🔍 Real-time search
- 📄 Pagination (12 per page)
- ➕ Business submission form

## Tech Stack
React 19 + TypeScript + Vite + Tailwind CSS + Supabase

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env`
3. Add your Supabase credentials
4. `npm install && npm run dev`

## Deploy on Netlify
Set these environment variables in Netlify:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
